About MailyDaily

MailyDaily is an Android app that integrates with Gmail to fetch unread emails, summarize them using AI, and suggest actionable items based on the email content.

Features

  • Google Sign-In for user authentication.
  • Email summarization and actionable recommendations.
  • Modern, user-friendly interface.

Setup Instructions

  1. Clone the repository and configure Firebase and Hugging Face API.
  2. Build and run the app using Android Studio.
